# Raspberry Pi Kiosk Setup

Scripts and instructions for configuring the Raspberry Pi.

## Requirements
- The Pi must boot up as quickly as possible.
- It must connect automatically to the ESP32's WiFi network (or utilize a wired internal bridge).
- It must launch a Chromium (or similar) browser in Kiosk mode, pointing to the ESP32's IP address.
- No desktop environment clutter, no screensavers, and no mouse cursor unless interacted with.
