# Audio DSP Firmware (Daisy)

Firmware for the Daisy Seed.

## Responsibilities
- Real-time audio processing (effects chain).
- SD Card management (loading impulse responses, presets, samples).
- MIDI processing (DIN/TRS MIDI In/Out).
- UART communication with the ESP32 to receive parameter updates.
