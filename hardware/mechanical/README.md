# Mechanical Design

This folder contains the mechanical assets for the ChimeraMultiFX pedal:
- 3D Models (STEP/STL) of the enclosure.
- Front panel/faceplate designs (e.g., SVG, DXF) showing drill holes for the 8 footswitches, 8 rotary encoders, and the 7" display.
- Mounting bracket designs for the internal PCB stack.
