# Audio DSP Board (Daisy Seed)

This board is responsible for high-fidelity audio processing.

## Requirements
- Host a Daisy Seed module.
- Stereo Audio In (1/4" Jacks).
- Stereo Audio Out (1/4" Jacks).
- MIDI In, Out, Thru (5-pin DIN or TRS).
- SD Card slot (accessible from the outside of the enclosure).
- UART headers for communication with the `control-esp32` board.
- Clean power input headers coming from the `power-supply` board.
