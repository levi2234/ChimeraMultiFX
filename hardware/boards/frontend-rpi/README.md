# Frontend Board (Raspberry Pi + LCD)

This board handles the physical mounting and routing for the UI.

## Requirements
- Mount for Raspberry Pi (e.g., Pi Zero 2 W).
- Mount/routing for 7" Touchscreen (DSI or HDMI + USB Touch depending on screen specs).
- Breakout connections to safely power the RPi from the `power-supply` board without browning out the audio.
